Now we are able to create multiple "Instances" of our class
First we put together all Datatypes and Functions that we want to put into our Class.
For proper Naming of our Class we take a look at the Codestyleguides for Python.
Then we start with the "Constructor" Function called __init__ with the

Self means the Object self or the Instance we create.
After that we add several Arguments that we want to process in our Class.
When we create a new instance of our class we assign the Arguments to their matching "Instance Variables"

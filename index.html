Lesson 1 - HTML and the World Wide Web
What is HTML?

The World Wide Web is a collection of HTML Documents.

HTML stands for Hypertext Markup Language.

The HTML Code inside these Documents is used to link many different Types of Information, like Text, Images, Video or other Media, together and create the so called “Web”.

HTML Document contain Content to display, Markup to apply formatting, references (links) to other Media or HTML Documents.

Within the past 25 Years this “Web” has grown to about 30 Billion “Webpages”

How can HTML Documents be viewed?

Most HTML Documents are stored on Webservers that are connected via the Wolrd largest Computer Network - the Internet.

With a Webbrowser Program running on your Computer,you should have a working Connection to the Internet, you request the transmisson of a HTML Document via the HTTP Protocol from the Server to your Computer.

HTTP stands for Hypertext Transfer Protocol.

After a successful transmission of the HTML Document, the Webbrowser reads the HTML Code insinde the Document and shows his interpretation of the HTML Code.

HTML Markup Basics

HTML Tags are used to create a Document.

An example Element in HTML begins with an ”opening” Tag, like <name> , than follows the content and ends with a "closing" Tag, like </name>

<h1>This is a Headingtext</h1>

All the three Parts of the Line above together form a HTML Element.

The type of the Element is defined by the Tag and tells the Browser how to display the Content. There are certain Tags that dont need an opening Tag, because of their special Function.

For example the </br> Tag. It defines a Linebreak, which don’t has content.

Lesson 2 - A Structured HTML Document
Blocks and Inline Elements

The example Element above is a so called Inline Element, it targets the content in between Start and Ending Tag directly.

Block Elements are used to define an invisible Block around the content

HTML Structure

HTML Documents are structured like a "tree". This means that Elements can have more Elements inside like in the follwing Example.

HTML Code Example:

  • <p>
    • <h1> Title </h1>
    • <b>
      • <em>Line 1</em> </br>
      • <em>Line 2</em> </br>
      • <em>Line 3</em> </br>
    • </b>
  • </p>

It’s recommended to write your HTML Code like the above, because it’s easier to understand when you have to change something.

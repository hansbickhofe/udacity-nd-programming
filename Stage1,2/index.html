Lesson 1 - HTML and the World Wide Web

What is HTML?

The World Wide Web is a collection of HTML Documents.
HTML stands for Hypertext Markup Language.
The HTML Code inside these Documents is used to link many different Types of Information, like Text, Images, Video or other Media, together and create the so called "Web".
HTML Document contain Content to display, Markup to apply formatting, references (links) to other Media or HTML Documents.
Within the past 25 Years this "Web" has grown to about 30 Billion "Webpages"

How can HTML Documents be viewed?

Most HTML Documents are stored on Webservers that are connected via the Wolrd largest Computer Network - the Internet.
With a Webbrowser Program running on your Computer,you should have a working Connection to the Internet, you request the transmisson of a HTML Document via the HTTP Protocol from the Server to your Computer.
HTTP stands for Hypertext Transfer Protocol.
After a successful transmission of the HTML Document, the Webbrowser reads the HTML Code insinde the Document and shows his interpretation of the HTML Code.

HTML Markup Basics

HTML Tags are used to create a Document.
An example Element in HTML begins with an "opening" Tag, like <name> , than follows the content and ends with a "closing" Tag, like </name>

<h1> This is a Headingtext</h1>

All the three Parts of the Line above together form a HTML Element.

The type of the Element is defined by the Tag and tells the Browser how to display the Content. There are certain Tags that dont need an opening Tag, because of their special Function.

For example the <br> Tag. It defines a Linebreak, which don’t has content.

Blocks and Inline Elements

The example Element above is a so called Inline Element, it targets the content in between Start and Ending Tag directly.

HTML Structure

HTML Documents are structured like a "tree". This means that Elements can have more Elements inside like in the follwing Example.

HTML Code Example:

  • <p>
    • <h1> Title </h1>
    • <b>
      • <em>Line 1</em> <br>
      • <em>Line 2</em> <br>
      • <em>Line 3</em> <br>
    • </b>
  • </p>

It’s recommended to write your HTML Code like the above, because it’s easier to understand when you have to change something.

Lesson 2 - Variables and Strings

A variable in Python

A variable defines a fixed name for a variable Value.
Example in Python:
fixed_name = 42
In the Example above the value of "42" is assigned to the variable "fixed_name".
This assingment is expressed with one single equal sign.
You can use those variables in further operations or expressions.
Also the value can be changed in further steps of your programm. In the above Example we used numeric Values. Alphanumeric content is called a string.
Simple strings are enclosed by single, or double quotation Marks.
Example:
fixed_name_for_string = "Test"
or
fixed_name_for_string = "Hello " + "Udacity"

Lesson 5 - Lists and For Loops

Lists

Lists can contain numbers, strings, numbers and strings mixed or other lists.
With lists we are are able to create structured Datasets.


Examples:
>>> numbers = [1,2,3,7.5]
>>> strings = ['Google','Yahoo','altavista',';-)']
>>> mixed = [0,8,1,5,'Otto Normal',['0815','007','4711',42]]

Listelements can be accessed like this:


>>> print numbers[3]
7.5
>>> print mixed[5][3]
42

Or sliced:


>>> print strings[2:]
['altavista', ';-)']

Mutation

Mutation on Lists means that if you can replace one ore more elements of it, without "touching" the other elements.


Example:
>>> letters = ['s','e','e']
>>> print letters
['s', 'e', 'e']
>>> letters[2] = 'a'
>>> print letters
['s', 'e', 'a']

Aliasing

Aliasing means that you can refer to an Object with a different Name and modifications assigned to the Aliasname are also done to the target Object.
If the original Variable content get a new assignment, the Aliasname still points to the Object that it was assigned too.


List Operations

.append - Method

With using the .append() Method on a List, you are able to add an element to the end of a List.


Example:
>>> letters = ['s','e','e']
>>> print letters
['s', 'e', 'e']
>>> letters.append('n')
>>> print letters
['s', 'e', 'e', 'n']
List addition

If two Lists are added with the '+' Operator a new list is created with the content of both lists.


Example:
>>> list1 = ['a','b','c']
>>> list2 = [1,2,3]
>>> list3 = list1 + list2
>>> print list3
['a', 'b', 'c', 1, 2, 3]
len()

The length (count of elements) of an List can be accessed with len(list)


Example:
>>> print len(numbers)
4
Loops on Lists

They return True or False.
